<h2>1. Metodología de trabajo en GIT</h2>
Se recomienda que cada desarrollador cree su propia rama a partir de la rama que desee trabajar.<br></br>
Es importante no alterar en la medida de lo posible las ramas principales.<br></br>

<div id='id2'/>
<h2> 2. Pasos para empezar a trabajar</h2>

1. Clona el repositorio completo en una carpeta de tu ordenador.
2. Ubicate en la rama donde desees trabajar.
3. Crea una rama de trabajo propia donde guardarás tus cambios.
4. Cuando termines los cambios, haz merge con la rama original de trabajo. Se enviará una pull request que deberá ser aceptada por un revisor.
